Lecture 2: HTML Structures

Time to learn how websites are structured with HTML!

Tools for Web Devs

Essential tools: a computer, text editor (like VSCode), browser (Chrome/Firefox), version control (GitHub), and optional tools like FTP or Figma.

Folder Structure

Your project folder should include:

  • index.html – main page
  • images/ – image files
  • styles/ – CSS files
  • scripts/ – JavaScript files

Basic HTML5 Structure

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <title>Page Title</title>
  </head>
  <body>
    Page content here
  </body>
</html>

HTML Elements

Everything in HTML is an element. Example:

<h1>Welcome</h1>

Has an opening tag, content, and closing tag.

Empty Elements

Some tags don’t wrap content. Example:

<br> – inserts a line break
<hr> – creates a horizontal rule

Attributes and Values

Attributes go in opening tags to describe elements.

<p>Centered text</p>

Nesting

Always properly nest tags. This is wrong:

<p><h1>Oops</p></h1>

Correct nesting avoids display issues.

Semantic Tags

Give meaning to your layout:

<header>, <nav>, <main>, <section>, <article>, <footer>

Use instead of non-semantic <div> and <span>.

Comments

Add notes to your HTML:

<!-- This is a comment -->

Helpful for organizing and debugging.

Text Formatting Tags

<b>Bold</b>
<strong>Strong</strong>
<i>Italic</i>
<em>Emphasized</em>

Use logical tags for better accessibility!

HTML Entities

Reserved characters need entities:

&lt; = <
&gt; = >
&amp; = &
&copy; = ©

Preformatted & Code

Use <pre> for preformatted blocks and
<code> for inline code examples.

Example:
<pre>
  <code>let x = 10;</code>
</pre>

Text Alignment

<p align="left">
<p align="center">
<p align="right">
<p align="justify">

Note: use CSS for modern alignment!

Save & View

Save your HTML file as .html or .htm
Open with a browser to preview your work!
