Lecture 4: CSS3 Fundamentals

Let's unlock the power of CSS3 and bring your HTML to life!

What is CSS?

CSS stands for Cascading Style Sheets and controls the presentation of HTML elements.

It separates content from design and allows styling to be reused across pages.

CSS Syntax

A CSS rule-set includes a selector and declarations.

selector { property: value; }
p { color: red; text-align: center; }

CSS Selectors

  • Element selector: p {}
  • ID selector: #para1 {}
  • Class selector: .center {}
  • Group selector: h1, h2, p {}

Three Ways to Insert CSS

  • External: <link rel="stylesheet" href="style.css">
  • Internal: <style> inside <head>
  • Inline: <h1 style="color:blue;">

CSS Comments

Used for documentation, ignored by browsers.

/* This is a comment */

CSS Color Properties

  • color – text color
  • background-color – background color
  • border-color – border color

Example:

<h1 style="color:Tomato;"
>Hello</h1>

Color Formats

  • rgb(255,0,0)
  • #ff0000
  • hsl(0, 100%, 50%)
  • rgba(255,0,0,0.5)

CSS Background

  • background-color
  • background-image
  • background-repeat
  • background-attachment
  • background-position

Shorthand:

background: #fff url("img.jpg") no-repeat top right;

CSS Borders

Style your borders:

  • border-style: solid, dashed, dotted, double
  • border-width: thin, 5px, etc.
  • border-color: red, #ff0000, rgb(255,0,0)
  • border-radius: rounded corners
p { border: 2px solid red; border-radius: 5px; }

Height & Width

Set element dimensions:

div { height: 100px; width: 50%; }
div { max-width: 600px; }
