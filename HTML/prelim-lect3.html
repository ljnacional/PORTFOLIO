Lecture 3: Text Fundamentals

Let’s dive into HTML text elements that add meaning and structure to your content!

<code> – Code Snippet

Displays a short piece of code in monospace font.

<code>let x = 5;</code>

Example: let x = 5;

<kbd> – Keyboard Input

Represents user input from a keyboard.

<kbd>Ctrl + S</kbd>

Example: Ctrl + S

<samp> – Sample Output

Used for sample output from a computer program.

<samp>File not found</samp>

Example: File not found

<u> – Underline

Underlines the text (use sparingly in modern web design).

<u>Important!</u>

Example: Important!

<ins> – Inserted Text

Indicates inserted or newly added text.

<ins>Updated info</ins>

Example: Updated info

&nbsp; – Non-Breaking Space

Inserts extra spacing that won’t break into a new line.

This  is  spaced

Example: This  is  spaced

HTML Entities

Special characters that must be written with an entity name/code:

  • &reg; – ® McDonalds®
  • &copy; – © CocaCola©

<blockquote> – Quotation

Indents quoted text from another source.

<blockquote>Quote here</blockquote>

Example:

"Lorem ipsum dolor sit amet, consectetur adipiscing elit..."

<div> – Generic Container

Used to group content for layout or styling purposes.

<div>This is a section</div>

<pre> – Preformatted Text

Preserves spaces and line breaks:

<pre>
  Line 1
  Line 2
</pre>

Example:

Line 1
    Line 2
        Line 3
