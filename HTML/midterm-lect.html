Lecture: Lists, Links & Images

Let's organize content with lists and connect the web with links and images!

Ordered Lists

Numbered lists for ordered items:

<ol>
  <li>First item</li>
  <li>Second item</li>
</ol>

Customize numbering with CSS:

ol {
  list-style-type: 
upper-roman;
}

Unordered Lists

Bulleted lists for items without order:

<ul>
  <li>Apples</li>
  <li>Bananas</li>
</ul>

Change bullet style:

ul {
  list-style-type: square;
}

Definition Lists

For terms and their definitions:

<dl>
  <dt>HTML</dt>
  <dd>HyperText Markup 
Language</dd>
  <dt>CSS</dt>
  <dd>Cascading Style 
Sheets</dd>
</dl>

Nested Lists

Lists within lists for hierarchy:

<ul>
  <li>Fruits
<ul>
    <li>Apples 
</li>
      <li>Oranges</li>
    </ul>
  </li>
  <li>Vegetables</li>
</ul>

Horizontal Lists

Create navigation menus:

ul.horizontal {
  list-style-type: none;
  margin: 0;
  padding: 0;
  overflow: hidden;
  background-color: #333;
}

li.horizontal {
  float: left;
}

li.horizontal a {
  display: block;
  color: white;
  text-align: center;
  padding: 16px;
  text-decoration: none;
}

Basic Links

The anchor tag creates hyperlinks:

<a href="https: 
//example.com">
Visit Example</a>

Link targets:

<a href="page.html" 
 target="_blank">
Opens in new tab</a>
<a href="#section-id">
Jumps to section</a>

Absolute vs Relative Links

Absolute links to other websites:

<a 
href="https://google.com">
Google</a>

Relative links within your site:

<a href="about.html"> 
About Us</a>
<a href="../contact.html"> 
Contact</a>

Link States

Style different link states:

/* Unvisited link */
a:link {
  color: blue;
}

/* Visited link */
a:visited {
  color: purple;
}

/* Mouse over link */
a:hover {
  color: red;
}

/* Selected link */
a:active {
  color: green;
}

Basic Images

The img tag displays images:

<img src="photo.jpg" 
 alt="Description">

Set image size:

<img src="logo.png" 
alt="Logo" 
     style="width:200px;
height:100px;">

Image Formats

Common web image formats:

  • JPEG - Best for photos
  • PNG - Best for transparency
  • GIF - Best for animations
  • WebP - Modern efficient format

Image Links

Make images clickable:

<a href="gallery.html">
  <img src="thumbnail.jpg" 
alt="Gallery">
</a>

Image Maps

Create clickable areas on images:

<img src="planets.jpg" 
alt="Planets" 
usemap="#planetmap">

<map name="planetmap">
  <area shape="rect" 
coords="0,0,82,126" 
        href="sun.html" 
alt="Sun">
  
<area shape="circle" 
coords="90,58,3" 
        
href="mercury.html" 
alt="Mercury">
</map>

Background Images

Add images to elements:

body {
  background-image: 
url("bg.jpg");
  background-repeat: 
 no-repeat;
  background-size: cover;
}

Floating Images

Wrap text around images:

<img src="cat.jpg" 
style="float:left; 
 margin:10px;">
<p>Text wraps 
around the image...</p>

Fantastic! Now you can organize content with lists and connect the web with links and images!
