Lecture 5: HTML Tables

Let's organize data with HTML tables!

Basic Table Structure

HTML tables are defined with the <table> tag:

<table>
  <tr>
    <th>Header 1</th>
    <th>Header 2</th>
  </tr>
  <tr>
    <td>Data 1</td>
    <td>Data 2</td>
  </tr>
</table>

<tr> = table row, <th> = table header, <td> = table data

Table Borders

Add borders with CSS:

table, th, td {
  border: 1px solid black;
}

For collapsed borders:

table {
  border-collapse: 
 collapse;
}

Table Styling

Cell padding and alignment:

th, td {
  padding: 15px;
  text-align: left;
}

Border spacing (only works without collapsed borders):

table {
  border-spacing: 5px;
}

Spanning Columns

Use colspan to merge cells horizontally:

<tr>
  <th colspan="2">
Merged Header</th>
</tr>
<tr>
  <td>Data 1</td>
  <td>Data 2</td>
</tr>

Spanning Rows

Use rowspan to merge cells vertically:

<tr>
  <th rowspan="2">Merged 
</th>
  <td>Data 1</td>
</tr>
<tr>
  <td>Data 2</td>
</tr>

Complete Table Example

<table style="width:100%; 
border-collapse:

Contacts</th>

    <td>Mike Johnson 
</td>

  </tr>

Table Captions

Add a caption to describe your table:

<table>
  <caption>
Monthly Sales</caption>
  <tr>
    <th>Month</th>
    <th>Sales</th>
  </tr>
  ...
</table>

Table Head, Body, Foot

Organize tables with semantic sections:

<table>
  <thead>
    <tr><th>Header</th>
</tr>
  </thead>
  <tbody>
    <tr><td>Data</td>
</tr>
  </tbody>
  <tfoot>
    <tr><td>Footer</td>
</tr>
  </tfoot>
</table>

Styling Tables

Use CSS to make tables more attractive:

table {
  width: 100%;
  border-collapse: 
 collapse;
}
th {
  background-color: 
 #f2f2f2;
}
tr:nth-child(even) {
  background-color: 
#f9f9f9;
}
tr:hover {
  background-color: 
#e9e9e9;
}
