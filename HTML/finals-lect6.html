Lecture 6: Forms & Multimedia

Let's create interactive forms and add multimedia to our web pages!

Form Basics

The <form> element creates interactive controls:

<form action="/submit"
 method="post">
  <!-- Form controls 
 go here -->
</form>

Key attributes:

  • action - Where to send form data
  • method - HTTP method (GET/POST)

Text Input

Basic text input fields:

<label for="name">
Name:</label>
<input type="text" 
id="name" 
       name="name" 
required>

Common attributes:

  • placeholder - Hint text
  • required - Mandatory field
  • maxlength - Character limit

Radio & Checkboxes

Single selection (radio buttons):

<input type="radio" 
id="male" 
       name="gender" 
 value="male">
<label for="male">
Male</label>

Multiple selection (checkboxes):

<input type="checkbox" 
 id="bike" 
       name="vehicle" 
value="bike">
<label for="bike">
Bike</label>

Dropdowns

Select elements for dropdown lists:

<label for="cars">
Choose a car:<
/label>
<select id="cars" 
 name="cars">
  <option value="volvo">
Volvo</option>
  <option value="saab">
Saab</option>
</select>

For multiple selections:

<select multiple 
size="3">
  <!-- options -->
</select>

Text Areas

Multi-line text input:

<label for="message">
Message:</label>
<textarea id="message" 
name="message" 
rows="4" cols="50">
</textarea>

Buttons

Submit and reset buttons:

<input type="submit" 
value="Submit">
<input type="reset" 
value="Reset">

Regular button with JavaScript:

<button type="button" 
onclick="alert('Hello!')">
  Click Me
</button>

HTML5 Input Types

Modern input types:

  • email - Email validation
  • date - Date picker
  • color - Color picker
  • range - Slider control
<input type="date" 
id="birthday" 
name="birthday">

Fieldset & Legend

Group related form elements:

<fieldset>
  <legend>Personal Info:<
/legend>
  <!-- Form controls -->
</fieldset>

Audio Element

Embed audio in HTML5:

<audio controls>
<source 
src="sound.mp3" 
type="audio/mpeg">
  Your browser 
 doesn't support audio.
</audio>

Supported formats: MP3, WAV, OGG

Video Element

Embed video in HTML5:

<video width="320" 
height="240" controls>
  <source 
src="movie.mp4" 
type="video/mp4">
  Your browser 
doesn't support video.
</video>

Supported formats: MP4, WebM, Ogg

Embedding Content

Using iframe for external content:

<iframe 
width="560" height="315" 
src="https://www.youtube
.com
/embed/
dQw4w9WgXcQ" 
  frameborder="0" 
allowfullscreen>
</iframe>

Form Best Practices

  • Always use <label> elements
  • Group related fields with <fieldset>
  • Use appropriate input types
  • Mark required fields clearly
  • Provide clear error messages
